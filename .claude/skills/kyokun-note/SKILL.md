# 教訓ノートの更新

今回のやり取りを振り返り、次回の依頼と作業を良くするための教訓を
LESSONS.md に追記・更新する。

## 書式（厳守）
- 1項目＝1教訓。冒頭に一行要約。
- 「失敗した頼み方 → 効いた言い直し」を対で書く。作業側の教訓は
  「失敗した進め方 → 効いた進め方」で書く。
- 既存の教訓と重複するなら、新規追加ではなく既存項目を更新する。
- 誤りと判明した教訓は削除する。
- リポジトリや会話履歴を見れば分かることは書かない。教訓だけを書く。

## 手順
1. LESSONS.md を読む。
2. 今回のセッションから教訓候補を挙げ、重複を照合する。
3. 追記・更新・削除を行い、変更点を一覧で報告する。
